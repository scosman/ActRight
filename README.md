# ActRight

ActRight is an AI-powered UI test workflow built on a simple thesis: AI should write and heal tests, not run them. It provides agent skills and conventions on top of Playwright Test -- AI agents author plain-English test descriptions and generate matching deterministic Playwright code, and separately heal those tests when the UI drifts. Running the suite is pure, fast, deterministic Playwright with no LLM in the loop.

Full documentation is coming in a future release. For now, see the specs in `specs/` for detailed design.
